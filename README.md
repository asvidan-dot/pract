# Математичні Алгоритми (Варіант 3)

Цей репозиторій містить реалізацію базових математичних алгоритмів на мові Python. Проєкт виконано в межах навчального завдання.

## 📋 Перелік реалізованих задач

1.  **Sieve of Eratosthenes (Easy)** — Пошук усіх простих чисел до заданого $n$.
2.  **Greatest Common Divisor (Easy)** — Знаходження найбільшого спільного дільника (рекурсивно та ітеративно).
3.  **Power Function (Medium)** — Швидке піднесення до степеня $O(\log n)$ з підтримкою від'ємних показників.
4.  **Square Root (Medium)** — Обчислення цілої частини квадратного кореня методом бінарного пошуку.
5.  **Factorial Trailing Zeroes (Easy)** — Визначення кількості нулів у кінці факторіала $n!$ без його повного обчислення.

---

## 🔬 Аналіз складності алгоритмів

| Задача | Алгоритм | Часова складність | Просторова складність |
| :--- | :--- | :--- | :--- |
| **Sieve of Eratosthenes** | Решето Ератосфена | $O(n \log \log n)$ | $O(n)$ |
| **GCD** | Алгоритм Евкліда | $O(\log(\min(a, b)))$ | $O(1)$ |
| **Power Function** | Binary Exponentiation | $O(\log n)$ | $O(1)$ |
| **Square Root** | Binary Search | $O(\log x)$ | $O(1)$ |
| **Trailing Zeroes** | Формула Лежандра | $O(\log_5 n)$ | $O(1)$ |
